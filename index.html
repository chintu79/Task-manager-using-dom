HTML Attributes vs DOM Properties

Attributes belong to the HTML markup, while properties belong to the DOM object created by the browser.

HTML Attribute

  • Defined in HTML
  • Retrieved using getAttribute()
  • Represents initial value
  • Stored in markup

DOM Property

  • Part of DOM object
  • Accessed using dot notation
  • Represents current value
  • Can change dynamically
